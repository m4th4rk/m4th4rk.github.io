Visão Geral do Projeto

Jogo Batalha Naval desenvolvido em C como projeto final da disciplina Introdução à Programação para Engenharias. Este projeto marca o início da minha jornada em desenvolvimento, demonstrando compreensão fundamental de algoritmos, estruturas de dados e lógica de programação.

O jogo implementa a clássica mecânica de Batalha Naval com interface de linha de comando, sistema de turnos e validação de movimento. Um projeto essencial para consolidar os conceitos básicos de programação.

Recursos Principais

Sistema de Tabuleiro

Implementação de matriz bidimensional para representar o tabuleiro de jogo com navios e acertos

Lógica de Programação

Uso de estruturas de controle, loops e validações para implementar regras do jogo de forma robusta

Sistema de Turnos

Implementação de alternância entre jogadores com rastreamento de estado e pontuação

Interface CLI

Interface de linha de comando clara e intuitiva para interação com o usuário
