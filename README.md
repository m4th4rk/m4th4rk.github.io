# Portfólio Pessoal - Matheus Marchi Baron

Este projeto é um portfólio pessoal desenvolvido como uma das atividades da disciplina **SSC0961 - Desenvolvimento Web e Mobile** no curso de Engenharia de Computação da **USP - São Carlos** (ICMC & EESC), cujo objetivo é trabalhar a prática de desenvolvimento web com HTML5, estilização com CSS3 e boas práticas de publicação.

## O Projeto
Foi adaptado o template **SnapFolio** (BootstrapMade) para que eu pudesse me apresentar, mostrando o meu trajeto acadêmico até o presente (abril de 2026), habilidades e interesses profissionais.

### O que foi usado?
* **Linguagens Base:** HTML5, CSS3, JavaScript.
* **Framework/Design:** Bootstrap 5 (Template SnapFolio).
* **Hospedagem:** GitHub Pages.
* **Apoio:** Foi utilizada Inteligência Artificial para fins didáticos, correção de bugs, sugestões e otimizações.
