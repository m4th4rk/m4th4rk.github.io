# Portfólio Pessoal - Matheus Marchi Baron

Este projeto é um portfólio pessoal desenvolvido como parte da disciplina **SSC0961 - Desenvolvimento Web e Mobile** no curso de Engenharia de Computação da **USP - São Carlos** (ICMC & EESC). O objetivo principal é consolidar conhecimentos práticos de estruturação web com HTML5, estilização com CSS3 e boas práticas de publicação.

## 🚀 Sobre o Projeto
O projeto utiliza como base o template **SnapFolio** (BootstrapMade), totalmente adaptado para refletir a minha trajetória acadêmica, habilidades técnicas (focadas em arquitetura e lógica) e minha grande paixão pelo desenvolvimento de jogos.

### Tecnologias e Ferramentas:
* **Linguagens Base:** HTML5, CSS3, JavaScript.
* **Framework/Design:** Bootstrap 5 (Template SnapFolio).
* **Hospedagem:** GitHub Pages.
* **Apoio:** O processo de personalização, limpeza de código desnecessário e adaptação estrutural contou com o auxílio de Inteligência Artificial para fins didáticos e otimização do workflow de desenvolvimento.

## 📝 Notas de Desenvolvimento
Este é um repositório com fins educativos e de apresentação profissional. As principais alterações e customizações realizadas no template original incluem:
* Reestruturação completa das seções "Sobre Mim" e "Currículo" para focar na trajetória de Engenharia e Game Dev.
* Implementação de avatares estilizados (Code Snippets) em substituição às imagens padrão.
* Configuração do sistema de filtros isotópicos no Portfólio para separar projetos da Graduação e Extracurriculares (Fellowship of the Game).
* Ajustes de responsividade, tipografia e adaptação do formulário de contato para envio direto via Formspree.
